# LogicHub

LogicHub is a browser-based prototype for creating and sharing logic maps.

## Features implemented

- Create maps
- Add nodes
- Connect nodes
- Drag nodes
- Zoom/pan editor viewport
- Edit node properties
- Fork maps
- Run simulations
- Feed with view/fork/comment/simulate actions

## Files

- `index.html` — UI layout
- `styles.css` — styling
- `logic-hub.js` — map model, editor logic, fork system, simulation engine, feed actions
- `ARCHITECTURE.md` — architecture and data model

## Run

Open `index.html` in any modern browser.
