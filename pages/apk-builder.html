The Local Compiling Mechanics

When you hit "Forge" or trigger the APK build pipeline, the local server passes your design variables to the **apk-builder** service running inside PM2.

The builder extracts a pre-configured, optimized webview boilerplate, generates assets, compiles client-side Javascript, and fires a headless Gradle build process mapping your Android SDK packages.

  • ✓ Uses local Java runtime environment (JDK 17)
  • ✓ Zero compile network requests to external servers
  • ✓ Output is a signed release APK ready for sideloading
