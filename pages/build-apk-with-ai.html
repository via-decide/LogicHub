Frequently Asked Questions

Can I run the compiled APK on any Android device?

Yes. The outputs are standard signed APK packages that can be sideloaded directly on any Android smartphone, tablet, or smart display.

Do I need Android Studio installed locally?

No. The local Mac Mini node runs a headless build environment with all necessary command-line tools configured out-of-the-box by LogicHub.
