# fixtures/kicad/smart-plant-pot

KiCad 7 fixture project for testing the kicad-adapter. Two variants exercise the diff engine's change detection:

## base/

Minimal sensor board with USB 5V input:
- J1: USB Micro power connector (5V + GND)
- U1: AMS1117-3.3 LDO (5V → 3.3V)
- C1: 10uF input decoupling cap
- C2: 100nF output decoupling cap
- U2: MCU sensor controller (4-pin)
- J2: 3-pin sensor connector (VCC/SENSE/GND)
- R1: 330R LED current limiter
- D1: Status LED

Nets: +5V, GND, +3V3, SENSE, LED, LED_A
PCB: 8 footprints, 15 tracks, board outline (gr_rect on Edge.Cuts)

## proposed/

Same project with controlled changes per spec section 14:
- **Replaced**: J1 (USB → battery connector), U1 (LDO → TPS62A02 buck regulator)
- **Added**: D2 (Schottky input-protection diode)
- **Removed**: R1 and D1 (status LED circuit)
- **BOM qty change**: C2 100nF quantity changes (C3 added)
- **Moved**: J2 sensor connector repositioned

## Generation

Files were generated by `generate-fixtures.mjs` with deterministic UUIDs derived from SHA-256 hashes. All files are valid KiCad 7 S-expression format with embedded lib_symbols (no external library dependencies).
